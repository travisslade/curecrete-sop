Open & set up background layers
1
Open transparent PNG from Blender

Layer menu → New Fill Layer → Solid Color. Simulates a light product listing background for QC.
3
Duplicate and set second solid color layer to #000000 new layer
Duplicate the light layer, double-click the color swatch, set to black. Rename to "BG — black" for clarity.
4
Drag both background layers beneath the product layer
Stack order from bottom: black BG → light BG → product PNG. Toggle visibility on each to switch QC backgrounds.
Build adjustment layer stack
5
Create Brightness & Contrast adjustment layer adj. layer
Clip to the product layer (Alt+click between layers) so it only affects the product, not the backgrounds.
6
Create Levels adjustment layer adj. layer
Clip to product layer. Bring in the black and white points — helps labels read cleanly without blowing out highlights.
7
Create Hue & Saturation adjustment layer adj. layer
Clip to product layer. Fine-tune color accuracy against the physical product or brand standards.
Adjust tone & QC
8
Dial in Brightness/Contrast, Levels, and Hue/Sat adjustments
Work iteratively. Prioritize label text legibility and accurate color over dramatic correction.
9
Toggle light background visible — check edits QC
Show the #f1f1f1 layer. Confirm label text is legible and product edges are clean with no fringe.
10
Toggle black background visible — check edits QC
Show the black BG layer. Look for halos, edge artifacts, and whether light-colored label text still reads clearly.
Smart object & highlights refinement
11
Convert product layer to Smart Object
Right-click the product layer → Convert to Smart Object. Enables non-destructive filter adjustments.
12
Apply Shadows/Highlights and adjust to improve text legibility
Image → Adjustments → Shadows/Highlights. Pull back blown highlights on the label surface so text stands out — especially where label color is close to the container color.
Export
13
Hide both background layers, export as transparent PNG PNG
Turn off both solid color layers before exporting. Use Export As to preserve alpha. Confirm the canvas shows as transparent in preview.

File naming reference
Convention: ItemCode(descriptor1)(descriptor2) — no spaces
Blender render
ItemCode(render)(v1)
Raw render out of Blender. Increment version number on re-renders.
Photoshop
ItemCode(PS)(v1)
Working PSD and transparent PNG export from Photoshop.
Final JPG
ItemCode(NetSuite)(product-image)
Final file sent to Garrett for upload. Match exactly to the item code in NetSuite.
